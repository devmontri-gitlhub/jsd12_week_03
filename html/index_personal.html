🚀 NEXT GEN IT

ชื่อเล่น: ตรี (Tri)

ชื่อจริง: มนตรี เทคโนโลยี (Montri Technology)

ผมเป็นนักพัฒนาไอทีรุ่นใหม่ (Next Generation IT) ที่มีความมุ่งมั่นในการพัฒนา ทักษะด้านเทคโนโลยีอย่างต่อเนื่อง มีความสามารถในการวิเคราะห์ปัญหา และออกแบบระบบให้มีประสิทธิภาพ พร้อมเรียนรู้เครื่องมือและเทคโนโลยีใหม่ ๆ เช่น Web Development, Database และ AI เพื่อสร้างผลงานที่ใช้งานได้จริง และตอบโจทย์ผู้ใช้งานในยุคดิจิทัล

ทักษะและสิ่งที่สนใจ

  • การเขียนโปรแกรมเว็บ (HTML, CSS, JS)
  • การออกแบบ UI/UX
  • ภาษาโปรแกรมที่อยากเรียน: Python, Go, Rust
  • งานอดิเรก: เล่นบอร์ดเกม, อ่านหนังสือเทคโนโลยี
